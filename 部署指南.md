# 🚀 微信文案助手 — 部署指南

跟着这 5 步走，大约 15 分钟搞定。

---

## 第 1 步：获取 DeepSeek API Key（花钱的一步）

1. 打开浏览器，访问 **https://platform.deepseek.com**
2. 点击右上角「注册」，用手机号或邮箱注册
3. 登录后，进入「**API Keys**」页面（左侧菜单）
4. 点击「**创建 API Key**」，随便取个名字（如 `wechat-polish`）
5. **复制并保存这个 Key**（只显示一次，丢了要重新生成）
6. 进入「**充值**」页面，先充 **10 元** 就够了（够用很久）

> 💰 DeepSeek 价格极低，日常使用一个月也就十块钱左右。

---

## 第 2 步：注册 GitHub（免费）

1. 打开 **https://github.com**
2. 点「Sign up」，用邮箱注册
3. 验证邮箱后登录

---

## 第 3 步：上传代码到 GitHub

你需要安装 **GitHub Desktop**（桌面版，操作最简单）：

1. 下载 GitHub Desktop：**https://desktop.github.com**
2. 安装并登录你的 GitHub 账号
3. 点击「**File → Add local repository**」
4. 选择 `C:\Users\Administrator\wechat-polish` 这个文件夹
5. 点「**create a repository**」，名字填 `wechat-polish`
6. 点「**Publish repository**」推送到 GitHub

---

## 第 4 步：部署到 Vercel（免费）

1. 打开 **https://vercel.com**
2. 点「Sign Up」，选择「**Continue with GitHub**」用 GitHub 账号登录
3. 授权后进入 Dashboard，点「**New Project**」
4. 在列表中找到 `wechat-polish`，点「**Import**」
5. 在部署配置页面：
   - Framework Preset 选 **Other**
   - 展开「**Environment Variables**」
   - Key 填 `DEEPSEEK_API_KEY`
   - Value 填你在第 1 步复制的那个 Key
6. 点「**Deploy**」按钮，等大约 1 分钟
7. 部署成功后，Vercel 会给你一个域名，类似：
   - `wechat-polish-xxxxx.vercel.app`

点击「Visit」就能看到你的网页了！

---

## 第 5 步：在微信里使用

1. 复制 Vercel 给你的域名（如 `wechat-polish-xxxxx.vercel.app`）
2. 打开微信，随便找个聊天窗口
3. 把链接发给自己（或发到文件传输助手）
4. 点击链接，微信内置浏览器就会打开你的文案助手
5. **建议把链接收藏** — 微信浏览器右上角「···」→ 收藏

---

## 使用方式

1. 输入你想说的话（比如「老板，方案我觉得不行」）
2. 选择风格（亲切随和 / 正式商务 / 幽默风趣 / 简洁直接）
3. 点「智能润色」
4. 复制结果，粘贴回微信聊天

---

## 遇到问题？

| 问题 | 解决方法 |
|------|---------|
| 点润色按钮没反应 | 检查 Vercel 环境变量 `DEEPSEEK_API_KEY` 是否正确 |
| 提示「AI 服务暂时不可用」 | DeepSeek 账户可能欠费了，去充值 |
| 微信里打不开链接 | 检查链接是否正确，确认 Vercel 部署成功 |
| 想改风格/加功能 | 回来找我，我帮你改代码然后重新部署 |

---

## 后续：改了代码怎么更新？

1. 在本地改好代码
2. 打开 GitHub Desktop，会看到修改的文件
3. 在左下角 Summary 写一行描述（如「改了风格」），点「Commit to main」
4. 点「Push origin」
5. Vercel 会自动检测到更新，自动重新部署（约 1 分钟）
6. 刷新链接即可看到新版本
